# CourseMatch
A course review platform for college students (prototype, currently under development).  
-  Designed only for Boston University students (planning to add more schools soon).

## Dev Tools
-  **Framework**: Flask (micro web framework written in Python)
-  **Front-end**: JS, HTML/CSS (Bootstrap4)
-  **DB**: SQLite3
-  **Other libraries**: jQuery, SocketIO, BeautifulSoup, ChartJS etc.
